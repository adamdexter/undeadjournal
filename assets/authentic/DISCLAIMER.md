# Archival assets — disclaimer

The files under this folder are the classic DeadJournal visual identity
(2003–2006 era): the skull header and footer bands, bone borders, parchment
backgrounds, stylesheet, favicon, navigation icons, and the "SK Cute Skulls"
mood icon set.

**Provenance.** All of these assets were sourced from public archives:

- the Internet Archive's Wayback Machine captures of `www.deadjournal.com`
  and `piktures.deadjournal.com` (captures from 2006–2013), and
- the public [Dreamwidth](https://github.com/dreamwidth/dreamwidth)
  open-source repository (the mood icon set).

**Copyright.** These works are included solely for historical preservation
and educational purposes, so that this time capsule remains usable even if
the archives it was recovered from become unavailable. All copyrights remain
with their rightful owners — DeadJournal.com / Warped, Inc. (site identity),
mindvamp (site design), and the mood theme's artist. This project makes **no
claim of ownership** over any of these works, derives no revenue from them,
and is not affiliated with, endorsed by, or connected to any rights holder.

**Takedown.** If you are a rights holder and want any of these files removed,
open an issue on this repository (or contact the maintainer directly) and
they will be removed promptly and without argument. The project functions
fully without this folder: `scripts/fetch-authentic-skin.sh` can fetch the
artwork from the public archives at install time instead, and
`LJ_SCHEME=bluewhite` in `.env` selects an artwork-free look.

**If you fork or redistribute this project**, this folder and this notice
travel together. Do not use these assets outside the context of a personal
DeadJournal-revival installation.
